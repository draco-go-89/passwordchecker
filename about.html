About
Simple UI. Heavy interaction. Password security for real humans.
What this tool does
  • Analyzes password strength with rule badges.
  • Estimates entropy + crack-time tiers.
  • Generates passwords from configurable character sets.
  • Keeps local history of your checks/generations.
Security note
This is a client-side educational tool. It does not guarantee real-world security. For best protection: use a password manager, unique passwords per site, and MFA.
